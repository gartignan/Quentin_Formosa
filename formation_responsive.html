Formation Responsive Web Design avec Bootstrap

Le Responsive Web Design sont les techniques permettant de concevoir un site adaptable pour toutes les appareil, mobile, tablet et ordinateur de bureau. Elles se fondent sur les media queries, et des framework comme Bootstrap.

  1. Introduction
    • Introduction au Responsive Web Design (exemples)
    • Principe et méthodes : Grille, Media Queries, Pourcentage
    • Présentation des différentes méthodes
    • Connaître son public d’écrans (étude statistiques)
  2. Les média queries
    • Syntaxe des média queries
    • Exemple de base des média queries
    • Positionner les éléments avec les média queries
    • Tester son design : émulateurs du navigateur, browserstack
    • Balise méta pour l’adaptation du viewport
  3. Les images
    • Positionnement d’image en background
    • Le format SVG
    • Les data-uri

  4. Bootstrap
    • Principe, téléchargement, installation

    • Manipulation de la grille : push, pull, offset etc.
    • Styliser les éléments : textes, tableau, boutons, icônes
    • Menus, Carrousel et Fenêtre modales
  • Durée de la formation : 2 à 3 jours
  • Prérequis : Connaître un peu de HTML et de CSS
