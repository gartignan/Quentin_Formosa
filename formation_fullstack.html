Formation Développeur FullStack

Cette formation d'un mois est pleinement personnalisable elle permet d'apprendre le métier de développeur Full Stack en 20 jours de formation.

  1. Base de données et SQL (2 jours)
    • Introduction
    • Langage d'interrogation des données
    • Langage de définition des données
    • Transactions
    • Fonction SQL
    • Triggers
    • MySQL, Posgre, Oracle ... : Quelles différences ?
  2. Java (3 jours)
    • Introduction
    • Syntaxe du Langage
    • Les tableaux
    • Les classes et les objets
    • Concepts avancés du langage
    • Les bibliothèques utiles
  3. Java EE (3 jours)
    • Introduction et Architecture
    • Web Services REST
    • Les WebSockets
    • Les EJB et JMS
    • Déploiement : Wildfly
  4. Hibernate (2 jours)
    • Introduction
    • Démarrage Rapide
    • Mapping Objet Relationnel
    • Opérateur CRUD, HQL et JQL
    • Hibernate avancé
  5. HTML5 et CSS3 (2 jours)
    • Introduction
    • Démarrage rapide
    • Principales balises
    • Mettre en forme les balises en CSS
    • Mettre en ligne un site web
  6. Responsive Web Design (1 jour)
    • Introduction
    • Les média queries
    • Les images
    • Bootstrap
  7. Javascript (3 jours)
    • Introduction
    • Environnement de débogage
    • Le langage Javascript
    • Les objets
    • Modification de Page et Evénements
    • Ajax
    • JQuery
  8. AngularJS (4 jours)
    • Présentation
    • Démarrage rapide
    • Concept de composants sous AngularJS
    • Binding et Classes
    • Boucle et Structures conditionnelles
    • Modifier le style
    • Maitriser les composants
    • Les services
    • Routage et Paramètre
    • Les Web Services
  • Durée de la formation : 20 jours
  • Prérequis : aucun
  • Lieu de la formation : Lyon
