Formation HTML5 / CSS

Le langage HTML5 et CSS3 permettent de concevoir, structurer et styliser un site web. De plus ils sont le fondement de nombreuses autres outils.

  1. Introduction
    • Les navigateurs web, le Standard W3C
    • Les résolutions d’écrans
    • Les étapes d’un projet Web
  2. Démarrage rapide avec le HTML5
    • Composition d’un site web
    • Balises et imbrications (Arbre de balises)
    • TP : Création d’une première page
  3. Principales Balises HTML5
    • Le langage, les principales balises
    • Sémantique des balises
    • Images et ImageMap

    • Balises Audio et Vidéo
    • TP : Création d’un livre interactif et d’un CV
  4. Mettre en forme les balises en CSS

    • Introduction aux serveurs web
    • TP : Mise en ligne des CV
    • Supplément : Etude des autres site web et introduction à bootstrap CSS
