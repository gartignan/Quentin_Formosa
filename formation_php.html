Formation PHP

Cette formation présente le developpement en PHP. Elle présente les notions du langage et le developpement dynamique de site web

  1. Introduction
    • Architecture Client Serveur
    • HTML, CSS, JavaScript et PHP : Présentation du SI
    • Protocole HTTP
    • Différence PHP4, PHP5 et PHP7
    • Exemple de Script PHP
  2. Le langage PHP
    • Commentaires
    • Les variables
    • Les fonctions
    • Les bibliothèques
    • Structures de contrôle
    • Structures conditionnelles
    • Les boucles
    • Les flux
    • Les Tableaux (Array)
    • Les Tableaux Associatifs
    • Les fonctions sur les tableaux
  3. Les formulaires
    • Formulaire en POST et GET
    • Passage des paramètres $_POST et $_GET
    • Exemple de login et mot de passe
  4. Les Sessions
    • Variables : Les cookies et les sessions
    • Les sessions sur le serveur
    • Exemple d'utilisation
    • Session et haute disponibilité
  5. Connexion à une base de données SQL et NoSQL
    • Présentation des bases de données
    • Présentation de MySQL.
    • Concepts fondamentaux : bases, tables, champs, enregistrements.
    • Exemple de connexion et de CRUD sur une base SQL(Create, Read, Update, Delete)
    • Présentation de MongoDB et de CassandraDB
    • Exemple de connexion et de CRUD sur une base NoSQL (Create, Read, Update, Delete)
