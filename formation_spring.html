Formation Spring & Spring Boot

Spring est probablement le Framework le plus populaire pour la conception de Web Service en Java. Ce framework en version 4 embarque un serveur et se nomme Spring Boot. La conception de web services et le developpement des tests devient simple et efficace.

  1. Introduction

    • Environnement de travail, installation des IDE

  2. Les WebServices

  3. Les Composants Spring

    • Repository

    • Création de Test Unitaire
    • Création de Test Integration
    • Pratiquer la TDD : Test Driven Development
